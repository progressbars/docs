Introduction

ProgressBar Tool is a Python library that makes it easy to create interactive progress bars in your terminal applications. It provides a simple API for displaying progress with customizable spinners, colors, estimated time remaining, callback functions, and support for multiple concurrent progress bars.

Key Features

Simple API

Easy-to-use interface that gets you started in minutes.

Customizable

Choose colors, spinners, and bar styles to match your app.

Multiple Bars

Support for managing multiple progress bars simultaneously.

ETA Calculation

Automatic estimated time remaining based on current progress.

Callback System

Execute custom functions when progress bars complete.

Thread Safe

Built-in thread safety for concurrent operations.

Quick Start

Get up and running with ProgressBar Tool in just a few lines of code:

from progressbar_tool import ProgressBar
import time

# Create a progress bar
bar = ProgressBar(total=50, prefix="Processing", spinner=True)
bar.start()

# Update progress
for i in range(51):
    bar.update(i, f"Step {i}/50")
    time.sleep(0.1)

bar.finish("Complete!")

This creates a progress bar with 50 total steps, displays a spinner, and shows progress updates with custom messages.

Multiple Progress Bars

Use MultiProgressBar to manage multiple progress bars simultaneously:

from progressbar_tool import MultiProgressBar
import time

mpb = MultiProgressBar()

# Add multiple progress bars
bar1 = mpb.add_bar(total=50, prefix="Download", color="green", spinner=True)
bar2 = mpb.add_bar(total=80, prefix="Process", color="blue", spinner=True)
bar3 = mpb.add_bar(total=60, prefix="Upload", color="yellow", spinner=True)

# Start all bars
mpb.start_all()

# Update all bars
for i in range(81):
    download_progress = min(i, 50)
    process_progress = min(i * 0.8, 80)
    upload_progress = min(i * 0.6, 60)
    
    mpb.update_all(
        [download_progress, process_progress, upload_progress],
        messages=[
            f"Downloading file {download_progress}/50",
            f"Processing data {int(process_progress)}/80",
            f"Uploading result {int(upload_progress)}/60"
        ]
    )
    time.sleep(0.03)

# Finish all bars
mpb.finish_all([
    "Download complete!",
    "Processing complete!",
    "Upload complete!"
])

ProgressBar Class

The main class for creating and managing progress bars.

Constructor

ProgressBar(total=100, prefix='', length=None, fill='‚ñà', empty='-', 
           color='green', spinner=False, spinner_style='default', 
           show_eta=True, message='')

Parameters

Parameter Type Default Description
total int 100 Total number of steps
prefix str '' Text before the progress bar
length int None Bar length (auto-detected from terminal width if None)
fill str '‚ñà' Character used for filled portion
empty str '-' Character used for empty portion
color str 'green' Color: 'green', 'yellow', 'red', 'blue', 'magenta', 'cyan'
spinner bool False Show spinner animation
spinner_style str 'default' Spinner style: 'default', 'dots'
show_eta bool True Show estimated time remaining
message str '' Initial message to display

Methods

start()

Start the progress bar timer and display the initial bar.

update(progress, message=None)

Update the progress bar with a new progress value.

  • progress (int): Current progress value (0 to total)
  • message (str, optional): New message to display

increment(step=1, message=None)

Increment the current progress by a specified amount.

finish(message='Done!')

Complete the progress bar and display a final message.

add_callback(func)

Add a callback function to be executed when the progress bar finishes.

MultiProgressBar Class

Manage multiple progress bars simultaneously.

Constructor

MultiProgressBar()

Methods

add_bar(**kwargs)

Add a new ProgressBar to the manager. Returns the created ProgressBar instance.

  • **kwargs: Any valid ProgressBar constructor parameters

start_all()

Start all progress bars in the manager simultaneously.

update_all(progresses, messages=None)

Update all progress bars with new progress values and optional messages.

  • progresses (list): List of progress values for each bar
  • messages (list, optional): List of messages for each bar

finish_all(messages=None)

Complete all progress bars with optional final messages.

  • messages (list, optional): List of final messages for each bar

Real World Usage

File Processing

from progressbar_tool import ProgressBar
import os
import time

def process_files(directory):
    files = [f for f in os.listdir(directory) if os.path.isfile(os.path.join(directory, f))]
    total_files = len(files)
    
    if total_files == 0:
        print("No files found in directory.")
        return
    
    bar = ProgressBar(
        total=total_files,
        prefix="Processing",
        color="blue",
        spinner=True,
        message=f"Found {total_files} files"
    )
    
    bar.start()
    
    for i, filename in enumerate(files):
        # Simulate file processing
        time.sleep(0.2)
        
        bar.update(
            i + 1,
            f"Processing {filename} ({i + 1}/{total_files})"
        )
    
    bar.finish(f"Processed {total_files} files successfully!")

# Usage
process_files("./my_files")

Data Processing Pipeline

from progressbar_tool import MultiProgressBar
import time

def data_pipeline():
    mpb = MultiProgressBar()
    
    # Add progress bars for each stage
    download_bar = mpb.add_bar(total=100, prefix="Download", color="green", spinner=True)
    process_bar = mpb.add_bar(total=150, prefix="Process", color="blue", spinner=True)
    upload_bar = mpb.add_bar(total=80, prefix="Upload", color="yellow", spinner=True)
    
    mpb.start_all()
    
    # Simulate pipeline stages
    for i in range(151):
        download_progress = min(i, 100)
        process_progress = min(max(0, i - 20), 150)
        upload_progress = min(max(0, i - 70), 80)
        
        mpb.update_all(
            [download_progress, process_progress, upload_progress],
            messages=[
                f"Downloading data {download_progress}/100",
                f"Processing records {process_progress}/150",
                f"Uploading results {upload_progress}/80"
            ]
        )
        time.sleep(0.05)
    
    mpb.finish_all([
        "Download complete!",
        "Processing complete!",
        "Upload complete!"
    ])
